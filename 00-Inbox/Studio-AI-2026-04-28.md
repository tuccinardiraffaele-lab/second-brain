# Studio AI — 2026-04-28

## TL;DR

- **Claude Code quality fix confermata** (v2.1.116, ~20 aprile): 3 bug distinti risolti — riduzione reasoning effort, session-clearing bug, system prompt verbosità. Chi usa Claude Code per sviluppo deve aggiornarsi.
- **Monitor tool** in Claude Code: nuovo strumento per streaming eventi da background scripts + subprocess sandboxing con PID namespace isolation su Linux. Potenzialmente rilevante per architetture agentiche.
- **Claude Opus 4.7 GA** (16 aprile): miglioramenti materiali su software engineering avanzato. Potenziale upgrade per agenti su task complessi.

**Dettaglio fix quality (v2.1.116 ~20 aprile):**
Tre problemi distinti identificati tra marzo e aprile 2026:
1. Riduzione intenzionale dello sforzo di reasoning per gestire latenza → mal ricevuta
2. Session-clearing bug → comportamento ripetitivo e perdita di memoria
3. Cambio system prompt per ridurre verbosità → danno collaterale su qualità coding

Tutti e tre revertiti/fixati. Fonte: postmortem Anthropic.
