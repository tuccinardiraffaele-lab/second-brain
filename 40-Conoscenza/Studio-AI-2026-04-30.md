# Studio AI — 2026-04-30

## TL;DR

- **POSTMORTEM Claude Code**: tre bug distinti risolti in v2.1.116 (20 apr) — il più critico cancellava il thinking di Claude ad ogni turno nelle sessioni lunghe (non solo dopo idle). Se [[Sistema-Agentico-Studio]] ha usato Claude Code nelle ultime settimane, le sessioni erano potenzialmente degradate. Ora risolto: aggiornare a v2.1.123+.

### Claude Code Changelog

**v2.1.123** (29 aprile):
- Fix: OAuth authentication che entrava in loop di retry 401 quando `CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` era impostato.

**v2.1.122** (28 aprile):
- **`ANTHROPIC_BEDROCK_SERVICE_TIER`**: nuova env var per scegliere il tier Bedrock (default/flex/priority) inviata come header. Rilevante solo per deploy su AWS Bedrock.
- **`/resume` + PR URL**: `/resume` ora accetta incolla di URL di PR (GitHub, GHE, GitLab, Bitbucket) e trova automaticamente la sessione che ha creato quella PR. Utile per il workflow di sviluppo [[Sistema-Agentico-Studio]].
- **`/mcp` migliorato**: mostra i connettori nascosti da server manuali con stesso URL, con hint per rimuovere duplicati.

**Versione attuale raccomandata: v2.1.123.**
