### Claude Code Changelog

**v2.1.126** (versione più recente — probabilmente rilasciata oggi o ieri):

Feature notevoli:
- **`claude project purge [path]`** — nuovo comando per rimuovere tutto lo stato di Claude Code (trascrizioni, task history) per un progetto. Utile per reset pulito di sessioni.
- **Model picker da gateway** — mostra i modelli disponibili dall'endpoint `/v1/models` del gateway quando `ANTHROPIC_BASE_URL` punta a gateway compatibili. Rilevante per setup con proxy/gateway custom.
- **`--dangerously-skip-permissions` esteso** — ora bypassa anche i prompt per scritture su directory protette (`.claude/`, `.git/`). Da usare con attenzione in script automatizzati.
- **OAuth paste-in-terminal** — OAuth login ora accetta codici incollati direttamente nel terminale quando il browser callback non può raggiungere localhost. Utile per ambienti remote/WSL.
- **Auto-downscale immagini** — immagini >2000px vengono automaticamente ridimensionate al paste. Riduce consumo token su paste di screenshot ad alta risoluzione.
- **Permission spinner rosso** — quando i check di auto-mode si bloccano, lo spinner diventa rosso come segnale visivo.
- Security fix e miglioramenti performance vari.

**Versioni precedenti (già coperte):**
- v2.1.123: fix OAuth 401 loop con `CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1`
- v2.1.122: `ANTHROPIC_BEDROCK_SERVICE_TIER`, `/resume` + PR URL, `/mcp` deduplication

**Versione raccomandata: v2.1.126.**
