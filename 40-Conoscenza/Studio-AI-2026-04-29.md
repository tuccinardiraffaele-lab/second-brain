### Claude Code Changelog — v2.1.121 (2026-04-28)

**Novità principali:**

- **`alwaysLoad` per MCP server config**: quando impostato a `true`, tutti gli strumenti di quel server MCP saltano il deferral della tool-search e sono sempre disponibili immediatamente. Semplifica le sessioni agentiche che dipendono da tool specifici disponibili dall'inizio.
- Fix: bug di doppia visualizzazione messaggi in terminali fullscreen.
- Fix: l'hint idle-return mostrava conteggio token errato.
- Fix: server MCP plugin bloccati su "connecting" all'avvio sessione.
- Miglioramento: velocità diff del tool Write su file grandi.

**ACP client v1.0.39**: toggle modalità allow-all via configurazione sessione. Aggiunti slash command `/compact`, `/context`, `/usage`, `/env`.

### Anthropic API — Breaking change (2026-04-30)

> [!warning] Ritiro beta 1M context — scade domani 2026-04-30
> Il beta header `context-1m-2025-08-07` cessa di funzionare su **Claude Sonnet 4** e **Claude Sonnet 4.5** domani. Le richieste che superano 200k token daranno errore.
>
> **Migrazione**: usare `claude-sonnet-4-6` o `claude-opus-4-6` — entrambi hanno 1M context window GA a pricing standard, nessun header beta necessario.
>
> **Verifica [[Sistema-Agentico-Studio]]**: controllare se qualche chiamata API usa ancora `anthropic-beta: context-1m-2025-08-07` su modelli Sonnet 4 o 4.5.
